# 逆天邪神模组说明

### 使用方法

解压后放置到`AmazingCultivationSimulator\Mods`文件夹下。

目录结构如下：

```
AmazingCultivationSimulator
|- Mods
   |- Evil
   |-	Settings
   |-	Info.json
   |- Readme.md
```

### 模组内容

1. 9个逆天邪神小说中的角色
2. 4本逆天邪神小说中的功法
3. 7大逆天邪神小说中的玄天至宝
4. 1把劫天诛魔剑
5. 3种用于提升技能的丹药
6. 1种用于提升五维的丹药
7. 4种土地改造
